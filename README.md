# MiniBank - Java EE CRUD Application

A banking system that demonstrates full CRUD operations using Java EE technologies.

## Technologies Used
- Java EE
- JPA (Java Persistence API)
- EJB (Stateless Session Beans)
- Servlets
- JSP
- GlassFish Server
- MySQL Database

## Features
- Create new bank account
- Search for account by ID
- Update account details
- Delete account
- View all accounts

## Project Structure
- `BankingSystemEJB` - Backend business logic and data persistence
- `BankingSystemWebApp` - Frontend web interface

## Setup Instructions
1. Install GlassFish Server
2. Set up MySQL database
3. Deploy the EJB module
4. Deploy the Web application
5. Access via browser at `http://localhost:8080/BankingSystemWebApp`
